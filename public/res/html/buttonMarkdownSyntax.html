Markdown语法

重点

*italic*   **bold**
_italic_   __bold__

链接

内联:

一个 [example](http://url.com/ "标题")

引用风格 (标题是可选的):

一个 [example][id]. 然后, 文档中的其他地方, 定义链接:
  [id]: http://example.com/  "标题"

图片

内联 (标题是可选的):

![alt text](/path/img.jpg "标题")

引用风格:

![alt text][id]

[id]: /url/to/img.jpg "Title"

标题

Setext风格:

标题 1
========

标题 2
--------

atx风格 (#标记可选闭合或不闭合):

# 标题 1 #

## 标题 2 ##

###### 标题 6

列表

有序, 没有段落:

1.  Foo
2.  Bar

无序, 带段落:

*   A列表项.

    多行段落.

*   Bar

可嵌套:

*   Abacus
    * answer
*   Bubbles
    1.  bunk
    2.  bupkis
        * BELITTLER
    3. burper
*   Cunning

引用

> Email-style angle brackets
> are used for blockquotes.

> > And, they can be nested.

> #### Headers in blockquotes
>
> * You can quote a list.
> * Etc.

内嵌代码

`<code>` 通过反斜点分隔.

You can include literal backticks
like `` `this` ``.

代码块

缩进4个空格或者1tab, 都会被渲染为代码块.

This is a normal paragraph.

    This is a preformatted
    code block.

水平分割线

3个横线或星号:

---

* * *

- - - -

手动换行

通过两个或者两个以上的空格换行:

Roses are red,
Violets are blue.
Based on the Markdown syntax guide, by Fletcher T. Penney.
